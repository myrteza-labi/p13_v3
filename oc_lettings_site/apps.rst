Détails de la classe
--------------------

**OCLettingsSiteConfig**  
Classe de configuration de l’application principale du projet.

- ``name`` : définit le chemin Python de l’application (``oc_lettings_site``).
- ``default_auto_field`` : utilise ``BigAutoField`` par défaut pour les clés primaires.
- ``verbose_name`` : nom lisible de l’application affiché dans l’admin Django.
