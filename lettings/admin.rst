Lettings – Interface d’administration
=====================================

Ce module configure l’affichage des modèles de l’application ``lettings`` dans l’interface d’administration Django.

.. automodule:: lettings.admin
   :members:
   :undoc-members:
   :show-inheritance:

Modèles enregistrés
-------------------

- **Address** : affiché avec tous ses champs dans l’interface d’admin.
- **Letting** : affiché également avec les champs définis dans son modèle.

Cela permet aux administrateurs de gérer les adresses et les locations directement via `/admin`.
