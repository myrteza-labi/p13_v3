Lettings – Configuration d'application
======================================

Ce module contient la configuration de l'application ``lettings``.

.. automodule:: lettings.apps
   :members:
   :undoc-members:
   :show-inheritance:

Classe principale
-----------------

**LettingsConfig**
~~~~~~~~~~~~~~~~~~

Classe de configuration Django pour l'application ``lettings``.

- ``name`` : Nom de l'application, utilisé par Django pour l'enregistrement.
- ``default_auto_field`` : Définit le type de clé primaire par défaut pour les modèles (BigAutoField).
